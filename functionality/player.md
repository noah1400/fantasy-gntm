players features.

players have access to player panel.

for each season
players can see their current points.
players can see a scoreboard of players.
players can see a leaderboard of models with their total points over all episodes
players can view a models details with points in each episode.

players get notifications as flash messages.
players get notification if draft starts.
players see the draft order.
players see which player is currently selecting a model.
players can see if it is their turn to select a model.
players can select a model out of all free models.
players can pick another model if one of their models get eliminated.
players can select a model that they want to remove if they have to.
players can swap a model if they are allowed to and the next player in order (see game logic)
